# Brief

## Purpose
Top Tools is a curated collection of essential development tools, extensions, and configurations for enhanced productivity.

## Core Requirements
- Document and maintain a curated list of VSCode/VSCodium extensions categorized by status (Active, Standby, Replaced, Deprecated)
- Document browser extensions for development, productivity, security, and accessibility
- Provide Git configuration files and best practices
- Store VSCode/VSCodium settings in a reusable JSON format
- List installed Windows programs with descriptions and performance impact
- Provide Cline AI assistant rules (AGENTS.md, WFGY.md) for behavioral guidelines and cognitive framework
- Curate personal bookmarks (fav.html files) for development resources
- Run XKCD daily scraper to fetch and save latest comic + explanation

## Scope
- Documentation-focused repository - no application runtime
- Configurations are meant to be copied and applied to personal development environment
- Memory Bank system tracks project evolution and technical decisions
